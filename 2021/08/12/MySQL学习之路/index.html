术语

  1. DB：DB（英文全称data base，数据库）是依照某种数据模型组织起来并存放二级存储器中的数据集合。
  2. DBMS：数据库管理系统(Database Management System)是一种操纵和管理数据库的大型软件，用于建立、使用和维护数据库。包括Oracle、MySQL等等软件。
  3. SQL：SQL 是用于访问和处理数据库的标准的计算机语言。
  4. 表：tables。是数据库中用来存储数据的对象，是有结构的数据的集合，是整个数据库系统的基础。
  5. 行：称为数据/记录（data）
  6. 列：称为字段（column）

SQL的分类：

  1. DQL（数据查询语言）：查询语句，凡是select语句都是DQL
  2. DML（数据操作语言）：insert delete update，对表中数据增删改
  3. DDL（数据定义语言）：create drop alter，对表的结构的增删改
  4. TCL（事务控制语言）：commit提交事务，rollback回滚事务
  5. DCL（数据控制语言）：grant授权，revoke撤销权限

比较运算符

比较结果为真，则返回 1，为假则返回 0，比较结果不确定则返回 NULL。

符号 描述 备注
= 等于
<>, != 不等于
> 大于
< 小于
<= 小于等于
>= 大于等于
BETWEEN 在两值之间 >=min&&<=max
NOT BETWEEN 不在两值之间
IN 在集合中
NOT IN 不在集合中
<=> 严格比较两个NULL值是否相等 两个操作码均为NULL时，其所得值为1；而当一个操作码为NULL时，其所得值为0
LIKE 模糊匹配
REGEXP 或 RLIKE 正则式匹配
IS NULL 为空
IS NOT NULL 不为空

执行顺序

select			5
...
from			1
...
where			2
...
group by		3
...
having			4
...
order by		6
...

所以，不可以在where里面使用分组函数，如下错误示例

寻找大于平均工资的所有员工：

select ename,sal from emp where sal > avg(sal);	//XXXXXXXXXXXXXXXXXX

那要怎么写呢？使用子查询：

select ename,sal from emp where sal > (select avg(sal) from emp);

having

找出每个部门薪资最高的，且大于2500：

mysql> select deptno,max(sal) from emp group by deptno having max(sal) > 2900;
mysql> select deptno,max(sal) from emp where sal > 2900 group by deptno;//效率较高

由于where先执行筛选，所以第二条效率更高

当然如果是求平均值且均值大于2000，则只能使用having：

mysql> select deptno,avg(sal) from emp group by deptno having avg(sal) > 2000;

自连接

inner join和on

一张表看成两张表

外连接

内连接和外连接的区别：

  • 内连接的两张表是平等的
  • 外连接中有主副之分。A表中的数据必须查出来，B表中的数据捎带着查出来

自连接：

select
    a.ename, b.ename
from
    emp a
inner join
    emp b
on
    a.MGR = b.empno;

左连接：

select
    a.ename, b.ename
from
    emp a
left join
    emp b
on
    a.MGR = b.empno;

右连接：

select 
    d.*
from
    dept d
left join 
    emp e
on 
    e.deptno = d.deptno
where
    e.empno is null;

查找员工的部门名称、工资等级、和上级领导

select 
    e.ename as "员工",d.dname,s.grade,e1.ename as "领导"
from 
    emp e
join
    dept d
on
    e.deptno = d.deptno
join 
    salgrade s
on
    e.sal between s.losal and s.hisal
left join
    emp e1
on
    e.mgr = e1.empno;
