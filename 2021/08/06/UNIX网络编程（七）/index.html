UNIX网络编程（七）

getsockopt和setsockopt

#include <sys/socket.h>   
int getsockopt(int sock, int level, int optname, void *optval, socklen_t *optlen);  
int setsockopt(int sock, int level, int optname, const void *optval, socklen_t optlen);

参数按顺序分别为：

  1. 套接字描述符
  2. 解释选项的代码，通用套接字的代码，某个特定协议的代码
  3. 需要访问的选项名。
  4. 指针。对于getsockopt(值-结果参数)，把已经获取的选项值存放在optval中。对于setsockopt(值参数)，从*optval中取得选项待设置的新值。
  5. 对于getsockopt()，作为入口参数时，选项值的最大长度。作为出口参数时，选项值的实际长度。对于setsockopt()，现选项的长度。

fcntl函数

#include <unistd.h>
#include <fcntl.h>

int fcntl(int fd, int cmd);
int fcntl(int fd, int cmd, long arg);         
int fcntl(int fd, int cmd, struct flock *lock);

fcntl函数提供了与网络编程相关的如下特性：

  • 非阻塞式IO。通过使用F_SETFL命令设置O_NONBLOCK文件状态标志，我们可以把一个套接字设置为非阻塞型。
  • 信号驱动IO。通过使用F_SETFL命令设置O_ASYNC文件状态标志，我们可以把一个套接字设置成一旦其状态发生变化，内核就产生一个SIGIO信号。
  • F_SETOWN命令允许我们指定用于接收SIGIO和SIGURG信号的套接字属主。

小结

介绍了四个函数：

  1. getsockopt
  2. setsockopt
  3. fcntl：把套接字设置为非阻塞型I/O或信号驱动式I/O的POSIX方法
  4. ioctl：后面章节介绍

由于选项这里的内容太繁杂，只探讨几个常用的选项。

  • SO_KEEPALIVE：周期性测试连接是否存活
  • SO_LINGER：若有数据待发送则延迟关闭
  • SO_RCVBUF：接收缓冲区大小
  • SO_SNDBUF：发送缓冲区大小
  • SO_REUSEADDR：允许重用本地地址
  • SO_BROADCAST和10个多播套接字选项仅适用于多播和广播
