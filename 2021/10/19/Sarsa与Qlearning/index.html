Sarsa与Qlearning

强化学习与博弈论 作业三

一、实验要求

  • 基于 Cliff Walk 例子实现 Sarsa 、 Q learning 算法
  • 提交一份文档包含算法实现核心思路、实验结果图等说明，不超过 4 页

二、实验原理

Q-Learning

首先要任意地初始化Q表（关于状态s和动作a的表）用以表示潜在的奖励。通常来说，只要全设为0就可以了。

每一次迭代，首先初始化当前状态。我们使用Epsilon greedy策略来选择一种行为（比如，在epsilon为0.9的时候，有90% 的情况我们会按照 Q 表的最优值选择一种行为, 10% 的概率随机地选择行为） 。

接着，我们要来更新Q表。要怎么更新呢？对于当前所作出的行为A，我们可以根据Q表查出它当前的潜在奖励——$Q(S,A)$，接着，我们可以根据下一个状态$S’$来预测出可能得到的奖励加上这次实实在在得到的奖励$R$。就可以得到现实与估计之间的差距$R+\gamma max_a Q(S’,a) - Q(S,A)$，其中$\gamma$为衰减值。我们将其乘以一个学习率$\alpha$。最后就可以得到更新后的$Q(S,A) \leftarrow Q(S,A) + \alpha[R + \gamma max_a Q(S’,a)-Q(S,A)]$

接着，我们就可以进入下一个状态$S’$，直到S为终点时，结束迭代。

Sarsa

Sarsa的整体过程与Q-Learning是类似的，不同的地方就在于：

Q-Learning说到不一定做到，是off-line learning，在学习的过程中当前状态使用了Epsilon greedy policy，但是获取下一个状态的时候却使用的是greedy policy。

而Sarsa则说到做到，是on-line learning，在学习的过程中当前状态使用了Epsilon greedy policy，在获取下一个状态的时候却使用的也是Epsilon greedy policy。

三、代码实现

1.前期准备

1.R表：记录了某个行为到达悬崖上的奖励为-100，其他地区的奖励为-1

R = np.zeros((ROW, COL, 4))
R[:, :, :] = -1.0
R[ROW-2, 1:11, 3] = -100.0
R[ROW-1, 0, 2] = -100.0

2.行为-状态转移表A：记录了状态$S_1$经过某个行为A后到达状态$S_2$

A = []
for r in range(0,ROW):
    A.append([])
    for c in range(0, COL):
        S = dict()
        #往左
        S[0] = [r, max(0, c-1)]
        #往上
        S[1] = [max(0,r-1), c]
        #往右
        S[2] = [r, min(COL-1,c+1)]
        #往下
        if c >= 1 and c <= COL-2 and r == 2:
            S[3] = [3,0]
        else:
            S[3] = [min(ROW-1, r + 1), c]
        A[-1].append(S)
A[ROW-1][0][2] = [ROW-1,0]

3.移动方向M表：0,1,2,3与左上右下一一对应

M = ['L','U','R','D']

2.Q-Learning

相关函数GreedyPolicy，跟据Q表来获取最大的预测回报。

def GreedyPolicy(x,y,q):
    max_q = q[x][y][0]
    max_a = 0
    for i in range(4):
        if q[x][y][i] >= max_q:
            max_q = q[x][y][i]
            max_a = i
    a = max_a
    return a

在GreedyPolicy的基础上，写出EpsilonGreedyPolicy：

def EpsilonGreedyPolicy(x,y,q,epsilon):
    #random.random()获取[0,1]之间的随机实数。
    if random.random() < epsilon:
        t = random.randint(0,3)
        a = t
    else:
        max_q = q[x][y][0]
        max_a = 0
        for i in range(4):
            if q[x][y][i] >= max_q:
                max_q = q[x][y][i]
                max_a = i
        a = max_a
    return a

3.Sarsa

核心代码：

def sarsa(q):
    for t in range(TIMES):
        for r in range(500):
            x = ROW-1
            y = 0
            a = EpsilonGreedyPolicy(x,y,q,EPSILON)
            while True:
                if x == ROW-1 and y==COL-1:
                    break
                [next_x,next_y] = A[x][y][a]
                reward = R[x][y][a]
                next_a = EpsilonGreedyPolicy(next_x,next_y,q,EPSILON)
                #核心公式
                q[x][y][a] += ALPHA*(reward + GAMMA*q[next_x][next_y][next_a]-q[x][y][a])
                x = next_x
                y = next_y
                a = next_a

4.Optimal policy

打印出结果，在每一个状态中选择reward最高的一个action：

def OptimalPolicy(q):
    for r in range(ROW):
        for c in range(COL):
            #使用*来表示悬崖
            if r == ROW-1 and 1 <= c <= COL-2:
                print("  *  ",end = "")
                continue
            a = GreedyPolicy(r,c,q)
            print(" ",M[a]," ",end = "")
        print("")
